- `*-dsar-log.csv` - registru DSAR istoric.
- `*-ropa.csv` - RoPA istoric.
- `*-vendor-dpa.csv` - registru furnizori si DPA-uri.
- `*-training.csv` - tracking training GDPR.
- `*-breach.csv` - breach log / ANSPDCP.
- `*-approval-history.csv` - aprobari istorice din email/Word.
- `*-evidence-archive.csv` - arhiva de dovezi din Drive.
- `template-dpa-vendor.md` si `template-dsar-access-response.md` - template-uri cabinet.

Regula: toate datele sunt pseudonimizate/fictive si pot fi folosite in browser local sau preview deploy pentru dry-run pre-pilot.
