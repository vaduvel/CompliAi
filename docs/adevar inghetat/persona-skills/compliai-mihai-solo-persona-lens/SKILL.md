# CompliAI Mihai / Solo Persona Lens

## Overview

Use this skill when reviewing copy, IA, onboarding, Home, Resolve, Cockpit, Dosar, Monitoring, or live runtime through Mihai's eyes.

Mihai wants:
- clarity
- first step
- case closure
- proof
- less anxiety

He does not want:
- jargon
- too many choices
- specialist pages in front
- hidden main actions

## Workflow

1. Name Mihai's current stage:
   - awareness
   - auth
   - onboarding
   - snapshot
   - resolve
   - cockpit
   - dosar
   - monitoring
2. Ask only these questions:
   - Does he understand in 5 seconds what this screen is for?
   - Does he see one dominant next action?
   - Is he being asked to think like a specialist?
   - Can he reach proof without wandering?
   - Will he feel safer after this step?
3. Mark the screen or flow as:
   - `clar pentru Mihai`
   - `obositor pentru Mihai`
   - `periculos pentru abandon`
4. If wrong, rewrite toward:
   - simpler wording
   - fewer choices
   - clearer next action
   - more obvious proof path

## Guardrails

- Do not optimize for expert delight.
- Do not let Mihai live in specialist modules.
- Do not accept dashboards with equal-weight cards.
- Do not keep context above action.

## Output standard

Return only:
- stage
- what Mihai wants here
- what blocks him
- what to simplify
- pass/fail verdict
