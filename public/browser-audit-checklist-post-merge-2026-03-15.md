## Intrebari obligatorii pe fiecare pagina

### 1. Prima actiune este clara?

- se intelege in 5-10 secunde:
  - ce faci aici
  - ce nu faci aici
  - care este primul pas corect

### 2. Intentia dominanta este unica?

- pagina pare clar una dintre:
  - orientare
  - executie
  - control
  - dovada
  - export
- nu pare doua sau trei pagini lipite la un loc

### 3. Summary / detail / action sunt in ordinea buna?

- sumarul vine primul
- detaliul vine dupa
- actiunea sau handoff-ul vine clar la final sau lateral
- nu exista concurenta egala intre toate

### 4. Handoff-ul este clar?

- userul vede unde continua dupa pagina curenta
- handoff-ul nu este ambiguu
- nu exista doua directii concurente fara explicatie

### 5. Rolul AI este corect?

- asistentul sau sistemul explica / propune
- omul valideaza
- nu apare impresia ca produsul decide singur

## Ce se marcheaza drept blocaj

- intentie de pagina neclara
- prima actiune neclara
- handoff rupt
- mixed intent evident
- suprafata vizibil prea grea pentru rolul ei

## Ce nu se marcheaza drept blocaj

- mici preferinte de spacing
- microcopy care poate fi polisat ulterior
- imperfectiuni vizuale minore daca flow-ul este deja clar
